Open-source and academic

Open-source projects and academic users get the “Pro” license + Deepkit Team Server for free. Write us an email with your setup and we reply quickly.
There are no additional fees. If you connect your own computation server to Deepkit Desktop App or to Deepkit Team Server you are obligated to pay all fees regarding that server or cloud provider. Deepkit is not responsible for any costs, damages or losses incurred.

All licenses include the full feature set of Deepkit although limited in its quota. For all paid licenses you will receive a license key from Deepkit which you need to enter into either the Deepkit Desktop app or the Deepkit Team Server administration interface. To activate the license it is required that an active internet connection is available.

For support requests regarding administration issues for the on-premises Deepkit Team Server it’s required to provide access to Deepkit employees via SSH. On-premises Deepkit Team Server is provided as Docker image and requires you to provide a functioning modern (not older than 1 year) Docker server engine on a Linux operating system.

Cloud storage and its API is provided under fair-use princinples. Deepkit has the right to limit access at any time if you exceed certain quotas regarding traffic or load. Deepkit informs you before taking actions.
